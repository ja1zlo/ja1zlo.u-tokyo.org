# 第1回 大学社団QSOパーティ 規約

## 主催

東京大学アマチュア無線クラブ  
学校社団一斉オンエア実行委員会

## 目的

大学など高等教育機関のアマチュア無線クラブの活動を活性化し、相互の交流を図る。

## 開催期間

2025年11月1日 から 2025年11月30日（日本標準時）

## 参加対象局

- **大学社団局**：大学や高等専門学校、短期大学など高等教育機関の社団局  
  （OB・OGは上記目的および趣旨をご理解の上ご参加ください。）
- **一般局**：大学社団局以外の個人局および社団局

## 周波数帯および電波形式

免許されているすべての周波数帯（WARCバンドを含む）および電波形式。

## 部門

- 大学社団部門  
  ※今回、一般部門の設置はありません。

## 呼び出し方法および交信内容

特に定めない。通常の交信と同様とする。

- 大学社団局は、可能であれば社団名や大学・学校名をアナウンスすること。
- 一般局は初心者に配慮した交信を行うこと。

## 得点およびマルチプライヤ

### 得点

コールサインとシグナルレポートの交換が完全に行われた交信を有効とし、以下の点数を与える。

| モード                                  | 得点 |
| :-------------------------------------- | :--: |
| SSB, FM など電話                        |  3   |
| CW（F2Aなどを含む）                     |  3   |
| FT8, FT4, JT-65などWSJT系デジタルモード |  1   |
| RTTY, SSTVなどその他のモード            |  3   |

- 同一日における同一バンドかつ同一モードでの同一局との交信は1回のみ得点とする。  
  （日付・バンド・モードのいずれかが異なれば、同じ局との交信でも得点となる）  
  例：同一局と同一バンドのFM・SSB・CW・FT8で交信した場合 → 10点。
- 他のコンテストや特定イベント中の交信は得点対象外。
- クロスモードでの交信は、使用モードのうち最も得点の低いものの得点とする。

### マルチプライヤ

得点を1点以上獲得した日を「運用日」とし、期間中の運用日数をマルチプライヤとする。

### 総得点の計算式

> > 総得点 ＝ 期間中の獲得ポイント合計 × マルチプライヤ(運用日数)

## 書類の提出

書類受付システム **ATS-4** へのアップロードをもって提出および受理とする。

- URL: [https://univoqp.allja1.org/](https://univoqp.allja1.org/)

### ログ提出における注意事項

1. その他のモードでのQSOは、すべてRTTYでログし提出すること。
2. クロスモード交信は得点の低い方のモードでログし提出すること。
3. 複数のコールサインを使用した場合は、代表コールサインにマージして提出すること。
4. ATS-4対応ログ形式（推奨）
   - ZLOXファイル（zLog）
   - ADIF
   - LG8（Ctestwin）
   - HDB（Turbo Hamlog）
5. 提出がうまくいかない場合は主催者まで連絡すること。

### 提出締め切り

2025年12月14日（日本時間）

## 個人情報の取り扱い

主催者は、収集した個人情報を当行事の運営および今後の行事告知のために使用する。

## 賞

| 部門         | 内容                                     |
| :----------- | :--------------------------------------- |
| 局数賞       | 最も多くの得点を獲得した局に賞状を贈る。 |
| 運用日数部門 | 運用日数が最も多かった局に賞状を贈る。   |
| 総合部門     | 総得点が最も多かった局に賞状を贈る。     |

## その他規約・禁止事項・失格事項

- レピータ、D-STAR、Wires-X等を使用した交信を認める。
- Webクラスタ等の使用を認める。
- アマチュア無線以外の手段（SNSなど）でのスケジュールQSOを認めるが、目的から逸脱しないこと。
- 大学社団局が異なるコールサインを用いることを認めるが、同一日・同一バンド・同一モードでの重複交信は1回のみ有効。
  同一大学社団内の異なるコールサイン同士の交信は得点にならない。
  - 例：東京大学アマチュア無線クラブ（JA1ZLO・JA1YWX）は通算可。ただしJA1ZLOとJA1YWX間の交信は得点にならない。
- 本規約に違反した場合、主催者が処分を決定する。
